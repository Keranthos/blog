## 🎨 **统一的设计语言**

### **颜色方案**
```css
主色调：#667eea → #764ba2 (紫色渐变)
文字：#333 (深灰)
次要文字：#666, #999 (灰色)
背景：white, rgba(255,255,255,0.95)
边框：#e8eaf0
```

### **圆角规范**
```css
小元素：8px, 12px (标签、小按钮)
中等元素：16px, 20px (卡片)
大元素：25px, 30px (输入框、大卡片)
超大元素：50px (导航栏)
```

### **阴影层次**
```css
轻阴影：0 4px 12px rgba(0,0,0,0.05)
中阴影：0 8px 30px rgba(0,0,0,0.12)
重阴影：0 20px 60px rgba(0,0,0,0.15)
```

### **动画参数**
```css
时长：0.3s - 0.6s
缓动：cubic-bezier(0.4, 0, 0.2, 1)
hover上移：translateY(-2px) 到 translateY(-10px)
```

## 🎨 **设计亮点**

### **1. 毛玻璃效果（Glassmorphism）** ✨
```css
background: rgba(255, 255, 255, 0.95);
backdrop-filter: blur(20px);
```

应用于：
- 导航栏
- 提问区浮动框
- 登录注册面板

### **2. 渐变色彩** 🌈
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

应用于：
- 激活的菜单项
- 所有按钮
- 品牌文字
- 用户头像
- 评论头像

### **3. 卡片化设计** 🎴
```css
background: white;
border-radius: 16px-20px;
box-shadow: 0 20px 60px rgba(0,0,0,0.1);
```

应用于：
- 文章内容
- 评论区
- 问题卡片
- 媒体卡片
- 登录表单

### **4. 微交互动画** 💫
```css
transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
```

应用于：
- 所有 hover 效果
- 下拉菜单
- 输入框聚焦
- 按钮点击
